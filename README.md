# Laravel 13 eCommerce API

Building a modern eCommerce API with **Laravel 13** and **PHP 8.5** focuses on high performance and strict type safety. By utilizing native **Backed Enums** for user roles, **Recursive Categories** for navigation, and **JSON-based storage** for product assets, this architecture ensures a scalable, developer-friendly shopping experience designed for the **2026 tech landscape**.

---

### Key Features
* **Laravel 13 Core**: Minimalist, high-speed framework architecture.
* **PHP 8.5+**: Leveraging property hooks and advanced type safety.
* **Dynamic Inventory**: Nested categories and soft-delete product management.
* **State-of-the-Art Auth**: Stateless security via Laravel Sanctum.
